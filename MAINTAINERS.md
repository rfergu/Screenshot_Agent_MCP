## How to Propose Changes

### Constitutional Amendment
1. Create pull request with proposed changes to `.specify/memory/constitution.md`
2. Include justification and impact analysis
3. Tag Rob Ferguson for review
4. Await approval from maintainer(s)

### Spec/Plan/Tasks Changes
1. Create pull request with proposed changes
2. Explain rationale (requirements change, discovered issue, etc.)
3. Update TRACEABILITY.md if applicable
4. Tag Rob Ferguson for review
